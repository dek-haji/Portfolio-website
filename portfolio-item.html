A short subtitle

Dek-overflow is a virtual note-taking react project app that makes it effortless for all developers to save their coding snippet, videos, and notes in one place.

Dek-Overflow is a virtual note-taking react project app that makes it effortless for all developers to save their coding snippet, videos, and notes in one place to save time instead of surfing google to find the information.The reason for building this app was to create a snippet management app and also build my snippet database. The Code Snippets Manager allows you to set the directories and individual snippets that you want available to insert into your code.
